# 코드 자동 수정 Agent

발생한 오류를 분석하고 자동으로 수정해주세요.

## 입력
$ARGUMENTS

## 수정 프로세스

### 1. 오류 분석
- 오류 메시지 파싱
- 관련 파일 및 라인 식별
- 근본 원인 파악

### 2. 수정 실행
- 최소한의 변경으로 오류 해결
- 기존 코드 스타일 유지
- 관련 import 문 자동 추가/수정

### 3. 검증
수정 후 반드시 `/check` 명령어를 실행하여 검증해주세요.

## 주의사항
- 불필요한 코드 변경 금지
- 기능 변경 없이 오류만 수정
- 수정 내용 요약 출력
